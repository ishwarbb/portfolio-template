XdoX

XdoX is an application that lets you start challenges, log your progress and show it off to the world through a unique profile page.

https://xdox.me AnishDe12020/xdox

It was March and my exams were over (and so I had also started 100 days of code) but I faced a big problem, that is, there was no "proper" way to log the 100 days of code progresses. I started out by logging it on a GitHub repository but found it cumbersome. Of course I had been tweeting my progress as well but eh...

Also the Hasura x Hashnode Hackathon had started so I found this as the perfect idea for a hackathon project.

Now, I regularly add my 100 days of code progress to XdoX, check it out on my profile page

For more details on XdoX, read my blog post introducing XdoX
